Sublime Text 搭建 LaTex 环境

本次latex环境主要涉及一下几个东西：

  1. Sublime Text 3.0 http://www.sublimetext.com/3
  2. miktex https://miktex.org/download
  3. SumatraPDF
    http://www.smart-soft.net/products/smart-pdf-converter/smart-pdf-converter.htm
  4. LatexTools

下载之后的文件如下图所示（其中SumatraPDF的名字已经被我重命名）

SublimeText的安装非常简单，本章不在赘述。miktex的安装也直接下一步直接点下去，注意：要记住miktex的安装路径，在后面配置环境是需要使用。

LatexTools的安装相比较有点麻烦，推荐手动安装，因为sublime text手动安装的话可能会由于网的问题出现许多问题。在sublime的插件包官网上搜过latextools下载具体可以参考下面的操作

打开ST->首选项->插件控制，会出现如图界面

点开搜索插件，进入下面界面：

在里面搜索自己想要的插件包，比如latextools

点进去会出现如下界面：

可以看出该插件被放在github仓库里面，点开图上所标记的链接，会进入一个github仓库中，接着可以进入repository进行插件包的下载。（记住包的路径）

注意 重点来了
